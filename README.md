# cl-tp3

SmartLibrary university lab implementing the Decorator Design Pattern in Java.

## Structure

- `smartlibrary/Loan.java`
- `smartlibrary/BasicLoan.java`
- `smartlibrary/LoanDecorator.java`
- `smartlibrary/ProtectionDecorator.java`
- `smartlibrary/DeliveryDecorator.java`
- `smartlibrary/ExtendedLoanDecorator.java`
- `smartlibrary/RareBookHandlingDecorator.java`
- `smartlibrary/Main.java`

## Run

```bash
javac smartlibrary/*.java
java smartlibrary.Main
```
